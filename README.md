# CHATBOT
Código fuente del desarrollo de un chatbot asistente diseñado para resolver dudas frecuentes y problemas comunes de los aspirantes a la Universidad del Magdalena. Este asistente conversacional tiene como propósito optimizar la atención y el acompañamiento durante el proceso de ingreso, brindando respuestas automáticas a consultas sobre fechas importantes, requisitos, documentación, inscripción, pruebas de admisión, y etc. El chatbot ha sido desarrollado utilizando tecnologías de procesamiento de lenguaje natural (PLN) e inteligencia artificial

Características principales:
Respuestas inmediatas a preguntas frecuentes.

Interfaz conversacional amigable con posibilidad de expansión a voz.

Sistema modular que permite escalar el número de respuestas o integrar nuevos módulos como seguimiento del proceso de inscripción, agenda de citas, entre otros.

Objetivo general:
Mejorar la experiencia de los aspirantes a través de un canal de atención 24/7 que reduzca la carga operativa del personal y facilite el acceso a la información institucional de forma clara y accesible.
